## 📌 Project Overview
This project provides a comprehensive analysis of the New York City Airbnb market. By examining over 49,000 listings, the goal was to identify the primary drivers of rental prices and provide actionable insights for property hosts to optimize their revenue.

# 🎯 Objectives
- Perform deep-dive **Exploratory Data Analysis (EDA)** on NYC borough trends.
- Clean and handle missing values in a real-world dataset.
- Visualize the relationship between **location, room type, and price**.
- Provide data-driven recommendations for host pricing strategies.

---

# 🛠️ Tech Stack
- **Language:** Python 3.8+
- **Libraries:** 
  - `Pandas` & `NumPy` for data manipulation.
  - `Matplotlib` & `Seaborn` for statistical visualizations.
  - `Scipy` for statistical testing.
- **Environment:** Jupyter Notebook

---

## 💡 Key Business Insights
*Location Premium:* Manhattan remains the most expensive borough, with an average price significantly higher than Brooklyn and Queens.
*   **Room Type Impact:** "Entire homes/apartments" make up the majority of listings and command a 60% higher price point than private rooms.
*   **Availability vs. Price:** Listings with high availability throughout the year tend to have lower prices, suggesting a "volume-over-margin" strategy for some hosts.
*   **Review Influence:** Neighborhoods with higher average reviews do not always correlate with higher prices, indicating that location is a stronger price driver than reputation.
