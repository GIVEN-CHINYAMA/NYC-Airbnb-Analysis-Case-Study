## 📌 Project Overview
This project provides a comprehensive analysis of the New York City Airbnb market. By examining over 49,000 listings, the goal was to identify the primary drivers of rental prices and provide actionable insights for property hosts to optimize their revenue.

## 🎯 Objectives
* Perform deep-dive **Exploratory Data Analysis (EDA)** on NYC borough trends.
* Clean and handle missing values in a **real-world dataset**.
* Visualize the relationship between **location, room type, and price**.
* Provide **data-driven recommendations** for host pricing strategies.

## 🛠️ Tech Stack
* **Language:** Python 3.8+
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scipy
* **Environment:** Jupyter Notebook

## 🚀 Quick Start
To replicate this analysis locally:
1. Clone the repo: `git clone https://github.com`
2. Install dependencies: `pip install pandas numpy matplotlib seaborn scipy`
3. Run `NYC_Airbnb_Analysis.ipynb` in your Jupyter environment.

## 💡 Key Business Insights
* **Location Premium:** Manhattan remains the most expensive borough, with an average price significantly higher than Brooklyn and Queens.
* **Room Type Impact:** "Entire homes/apartments" command a **60% higher price point** than private rooms.
* **Availability Strategy:** Listings with high availability tend to have lower prices, suggesting a **"volume-over-margin"** strategy for some hosts.
* **Review Influence:** Location is a **stronger price driver** than reputation; neighborhoods with high reviews don't always correlate with higher prices.
